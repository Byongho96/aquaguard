## 배경지식
- Flutter 프로젝트는 트럭(호차)에서 운용되는 시스템. React 프로젝트는 여러 호차에서 들어오는 데이터를 처리하는 관제 프로젝트.
- 그러나 현재는 데모버전이기 때문에 트럭이 하나밖에 없어서, React 프로젝트에서도 Flutter와 마찬가지로 하나의 트럭에 대한 데이터를 처리
